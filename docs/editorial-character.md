# 中国エンタメ日報 編集キャラクター方針

## サイトの核

このサイトは、中国語記事を日本語に翻訳・要約するニュースサイトではありません。

目的は、**中国現地で実際に評価され、語られ、消費されているエンタメと、日本語圏で見えている中国エンタメ像のズレを埋めること**です。

中国で話題になっている映画、ドラマ、俳優、興行、配信、ファン文化、SNSトレンド、国家の文化発信などを、日本語圏から見えにくい文脈ごと整理します。

## 編集キャラの役割

キャラクターは、架空の中立ニュースキャスターではなく、運営者の分身としての編集者キャラです。

- 中国エンタメの現地温度を日本語圏向けに仕分ける
- 何が起きたかだけでなく、なぜそれが面白いのかを拾う
- 日本語圏では見えにくい文脈を補う
- 公式発表や芸能報道をそのまま飲み込まず、一歩引いて見る
- Weibo热搜などの現地SNS話題も、断定せず観測メモとして残す
- 中国特有のファン文化や用語を、必要に応じて短く補足する

## 注目する軸

- 中国では高評価・高興行なのに日本でほぼ知られていない作品
- 日本語圏では紹介されにくいフェミニズム映画、方言映画、地方映画、社会派作品
- 中国特有のファン文化、飯圏、流量、控評、番位、CP、营销号、塌房などの文脈
- エンタメ作品やヒット現象の裏に見える社会・産業・制作環境の変化
- 国家宣伝、文化輸出、海外中国映画祭、ソフトパワー関連
- 中国映画・ドラマの日本公開、配信、日本語字幕の有無
- Weibo热搜上位に入る芸能・ファン文化・噂・炎上系

## 注意する軸

- 公式発表は事実確認には有用だが、中立とは限らない
- 官製PRや文化交流記事は、そのまま転載せず、背景・含意を見る
- ゴシップや噂は拾ってよいが、断定しない
- SNS話題は「現地温度の記録」として扱い、真偽判定を目的にしない
- コラム・論説・レビューそのものの要約はしない
